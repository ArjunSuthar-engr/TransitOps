# Backend API

This is the Express backend for the SERN stack hackathon project.

## Structure
- `src/config/`: Configurations (database, server, etc.)
- `src/controllers/`: Handle requests/responses
- `src/middleware/`: Custom Express middlewares
- `src/models/`: Database models/queries
- `src/routes/`: Express router definitions
- `src/services/`: Business logic and external API integrations
- `src/utils/`: Helper utilities and shared functions
- `src/app.js`: Configures the app, routing, and middlewares
- `src/server.js`: Starts the server on a specified port
